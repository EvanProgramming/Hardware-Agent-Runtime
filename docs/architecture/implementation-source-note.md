# Implementation source note

The implementation brief refers to `docs/architecture/`, `docs/architecture/adr/`, `examples/drivers/`, and `examples/experiments/`. The completed architecture phase instead stored its approved contracts in the root `docs/` files and `schemas/har-v1.schema.json`; the referred directories were absent at implementation start.

This is a documentation-location mismatch, not an architectural contradiction. The existing root documents remain normative. This directory is introduced only for implementation notes and future ADR expansion; it does not replace or alter the approved boundaries, data contracts, or ADR decisions.
